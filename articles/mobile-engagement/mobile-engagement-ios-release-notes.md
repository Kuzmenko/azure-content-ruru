#Заметки о выпуске пакета SDK для платформы Azure Mobile Engagement (iOS)

##4\.0.0 (12.09.2016)

-   Исправлена ошибка, из-за которой не приводились в действие уведомления на устройствах iOS 10.
-   Устаревшая версия XCode 7 заменена новой версией.

##3\.2.4 (06/30/2016)

-   Исправлено агрегирование данных из технических и других журналов.

##3\.2.3 (07.06.2016)

-   Исправлена ошибка, из-за которой не сообщалось о доставке отзыва, когда приложение работало в фоновом режиме.
-   Оптимизирована отправка технических журналов.

##3\.2.2 (07.04.2016)

-   Исправлена ошибка при отмене HTTP-запроса, которая иногда приводила к сбою.

##3\.2.1 (11.12.2015)

-   Фиксированная задержка при активации нового экземпляра приложения уведомлением с прямыми ссылками.

##3\.2.0 (08.10.2015)

-   В пакете SDK включен Bitcode для работы с **Xcode 7**.
-   Исправлены ошибки, связанные с уведомлениями в приложении.
-   Уведомления в приложении стали более надежными при низком уровне заряда аккумулятора и в других подобных ситуациях.
-   Удалены лишние журналы консоли, созданные библиотекой сторонних производителей.

##3\.1.0 (26.08.2015)

-   Исправьте ошибку совместимости iOS 9 с библиотекой стороннего производителя. Она вызывала сбои при отправке результатов опросов, сведений о приложении или дополнительных данных.

##3\.0.0 (06/19/2015)

-   Mobile Engagement использует автоматические push-уведомления.
-   Прекращена поддержка iOS 4.X. Начиная с этой версии, целевое устройство для развертывания приложения должно быть с версией не ниже iOS 6.

##2\.2.0 (05/21/2015)

-   В основе идентификатора устройства Mobile Engagement для устройств c версией ОС ниже iOS 6 теперь используется GUID, генерируемый во время установки.

##2\.1.0 (24.04.2015)

-   Добавлена поддержка Swift.
-   При щелчке уведомления теперь после открытия приложения вызывается URL-адрес действия.
-   В пакет SDK добавлен отсутствующий файл заголовка.
-   Исправлена проблема, при которой отключалось средство создания отчетов об аварийном завершении Mobile Engagement.

##Версия 2.0.0 (17.02.2015)

-   Первый выпуск Azure Mobile Engagement
-   Конфигурация appId/sdkKey заменена конфигурацией строки подключения.
-   Удален API для отправки и получения произвольных сообщений XMPP из произвольных сущностей XMPP.
-   Удален API для отправки и получения сообщений между устройствами.
-   Улучшения безопасности.
-   Удалено средство отслеживания SmartAd.
