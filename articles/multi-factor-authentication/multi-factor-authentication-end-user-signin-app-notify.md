# Вход в мобильные приложения с помощью уведомлений службы Azure Multi-Factor Authentication

Ниже описана процедура многофакторной проверки подлинности для входа в мобильное приложение с помощью отправленных уведомлений.

## Вход с помощью уведомления, отправленного в мобильное приложение

<ol>

<li>Войдите в приложение или службу, например Office&#160;365, используя имя пользователя и пароль.</li>
<li>Ожидайте уведомления от корпорации Майкрософт.</li>


<center>![Установка](./media/multi-factor-authentication-end-user-signin-app-notify/notify.png)</center>

<li>Ответьте на него и нажмите кнопку подтверждения.</li>

<center>![Установка](./media/multi-factor-authentication-end-user-signin-app-notify/phone.png)</center>


<li>После этого вы войдете в систему.</li>
