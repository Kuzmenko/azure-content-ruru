# Создание и удаление паролей приложений на портале Office 365 для службы Azure Multi-Factor Authentication

Если вы пользуетесь многофакторной проверкой подлинности в Office 365, вам необходимо будет создавать и удалять пароли приложений на портале Office 365.

## Для создания паролей приложений на портале Office 365
--------------------------------------------------------------------------------

<ol>
<li>Войдите на портал Office&#160;365.</li>
<li>В правом верхнем углу выберите мини-приложение параметров.</li>
<li>Слева выберите "Дополнительная проверка безопасности".</li>

![Облако](./media/multi-factor-authentication-end-user-app-passwords-create-o365/security.png)

<li>Справа выберите **Обновить мои телефонные номера, используемые для обеспечения безопасности учетной записи**.</li>
<li>На странице подтверждения вверху выберите пароли приложений</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/addsec.png)</center>

<li>Нажмите кнопку **Создать**.</li>


<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/apppass.png)</center>

<li>Введите имя для пароля приложения и щелкните **Далее**.</li>
<li>Скопируйте пароль приложения в буфер обмена и вставьте его в свое приложение.</li>



## Удаление паролей приложений на портале Office 365
--------------------------------------------------------------------------------

<ol>
<li>Войдите на портал Office&#160;365.</li>
<li>В правом верхнем углу выберите мини-приложение и щелкните "Параметры Office&#160;365"</li>
<li>В нижней части щелкните "Дополнительная проверка безопасности" и выберите **Обновить мои телефонные номера, используемые для обеспечения безопасности учетной записи**.3</li>

![Облако](./media/multi-factor-authentication-end-user-app-passwords-create-o365/security.png)

<li>Справа выберите **Обновить мои телефонные номера, используемые для обеспечения безопасности учетной записи**.</li>
<li>На странице подтверждения вверху выберите пароли приложений</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/addsec.png)</center>

<li>Рядом с паролем приложения, который следует удалить, выберите **Удалить**.</li>


<center>![Cloud](./media/multi-factor-authentication-end-user-app-passwords-create-o365/apppass.png)</center>

<li>Появится всплывающее окно с предложением подтвердить удаление.</li>
<li>Щелкните **Да**.</li>
