# Использование мобильного приложения для связи при работе с Многофакторной Идентификацией Azure

Из этой статьи вы узнаете, как использовать приложение Microsoft Authenticator в качестве основного способа связи. Здесь описана настройка Многофакторной Идентификации Azure с использованием мобильного приложения в качестве основного способа связи.

Приложение Microsoft Authenticator доступно для платформ [Windows Phone](http://go.microsoft.com/fwlink/?Linkid=825071), [Android](http://go.microsoft.com/fwlink/?Linkid=825072) и [iOS](http://go.microsoft.com/fwlink/?Linkid=825073).

## Использование Microsoft Authenticator в качестве способа связи


1. На экране **Дополнительная проверка безопасности** из раскрывающегося списка выберите **Мобильное приложение**.
2. Выберите **Уведомление** или **Одноразовый пароль** и нажмите кнопку **Установить**.

	![Экран дополнительной проверки безопасности](./media/multi-factor-authentication-end-user-first-time-mobile-app/mobileapp.png)

3. Откройте установленное на телефоне приложение Microsoft Authenticator и нажмите **+**, чтобы добавить учетную запись.
4. Укажите, что вы хотите добавить рабочую или учебную учетную запись. При этом откроется сканер QR-кода. Если камера на телефоне неисправна, то можно вручную ввести сведения о компании. Дополнительные сведения см. в разделе [Добавление учетной записи вручную](#add-an-account-manually).

	![Экран выбора учетной записи](./media/multi-factor-authentication-end-user-first-time-mobile-app/scan.png)

	![Экран, в котором выбирается считывание QR-кода или ввод данных учетной записи вручную](./media/multi-factor-authentication-end-user-first-time-mobile-app/scan4.png)

5. Считайте изображение QR-кода, которое отображается при открытии экрана настройки мобильного приложения. Нажмите кнопку **Готово**, чтобы закрыть экран с QR-кодом.

	![Экран QR-кода](./media/multi-factor-authentication-end-user-first-time-mobile-app/scan2.png)

	Если не удается считать QR-код, то сведения можно ввести вручную.

	![Экран ввода информации вручную](./media/multi-factor-authentication-end-user-first-time-mobile-app/barcode.png)

6. После выполнения активации на телефоне выберите **Связаться со мной**. При этом на ваш телефон будет отправлено уведомление или код проверки. Нажмите кнопку **Проверить**.

	![Экран, где для входа необходимо нажать кнопку "Проверить"](./media/multi-factor-authentication-end-user-first-time-mobile-app/verify.png)

7. Если в вашей компании для утверждения входа требуется ПИН-код, введите его.

	![Поле для ввода ПИН-кода](./media/multi-factor-authentication-end-user-first-time-mobile-app/scan3.png)

8. После ввода ПИН-кода нажмите кнопку **Закрыть**. На этом этапе проверка будет считаться выполненной.
9. Рекомендуется указать номер своего мобильного телефона на случай утери доступа к мобильному приложению. Выберите соответствующую страну в раскрывающемся списке и введите номер мобильного телефона в соседнее поле. Щелкните **Далее**.
10. Вы уже настроили способ связи. Теперь необходимо задать пароли для приложений, не использующих браузер, таких как Outlook 2010 или более ранних версий. Если вы не используете такие приложения, нажмите кнопку **Готово**. В противном случае перейдите к следующему шагу.

	![Экран создания пароля приложения](./media/multi-factor-authentication-end-user-first-time-mobile-app/step4.png)

11. Если вы работаете с приложениями, не использующими браузер, скопируйте предоставленный пароль и вставьте его в свое приложение. Действия для конкретных приложений, включая Outlook и Lync, см. в разделах "Как сменить пароль электронной почты на пароль приложения" и "Как изменить пароль в приложении на пароль приложения".
12. Нажмите кнопку **Готово**.


## Добавление учетной записи вручную
Для добавления учетной записи вручную выполните описанные ниже действия.

1. Нажмите кнопку **Enter account manually** (Ввести учетную запись вручную).

	![Экран ввода кода и URL-адреса](./media/multi-factor-authentication-end-user-first-time-mobile-app/addaccount.png)

	![Экран ввода кода и URL-адреса](./media/multi-factor-authentication-end-user-first-time-mobile-app/addaccount2.png)

2. Если у вас есть учетная запись, для которой уже настроена Многофакторная Идентификация Azure, то введите код и URL-адрес, указанные на той же странице, что и штрихкод. Эти данные необходимо ввести в поля **Код** и **URL-адрес** в мобильном приложении.

	![Настройка](./media/multi-factor-authentication-end-user-first-time-mobile-app/barcode2.png)

3. По завершении активации выберите **Связаться со мной**. При этом на ваш телефон будет отправлено уведомление или код проверки. Нажмите кнопку **Проверить**.
