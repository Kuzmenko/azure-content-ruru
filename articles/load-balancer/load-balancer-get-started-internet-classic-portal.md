## Настройка подсистемы балансировки нагрузки, доступной в Интернете, для виртуальных машин

Чтобы распределить нагрузку сетевого трафика из Интернета между виртуальными машинами облачной службы, нужно создать набор балансировки нагрузки. В этой процедуре предполагается, что вы уже создали виртуальные машины и что все они находятся в одной облачной службе.

**Настройка набора балансировки нагрузки для виртуальных машин**

1. На классическом портале Azure щелкните **Виртуальные машины**, а затем выберите имя виртуальной машины в наборе балансировки нагрузки.

2. Щелкните **Конечные точки**, а затем выберите **Добавить**.

3. На странице **Добавление конечной точки к виртуальной машине** нажмите стрелку вправо.

4. На странице **Укажите сведения о конечной точке** введите следующую информацию.

    * В поле **Имя** введите имя конечной точки или выберите имя в списке предварительно определенных конечных точек для распространенных протоколов.
    * В поле **Протокол** выберите при необходимости протокол, который требуется для этого типа конечной точки (TCP или UDP).
    * В полях **Общий порт** и «Частный порт» при необходимости введите номера портов, которые требуется использовать для виртуальной машины. На виртуальной машине можно использовать частный порт и правила брандмауэра, чтобы перенаправить трафик способом, который подходит для вашего приложения. Частный порт может быть таким же, как общий порт. Например, для конечной точки для веб-трафика (HTTP) для общего и частного порта можно назначить порт 80.

5. Выберите **Создать набор балансировки нагрузки** и щелкните стрелку вправо.

6. На странице **Настроить набор балансировки нагрузки** введите имя для набора балансировки нагрузки, а затем присвойте значения для поведения проб балансировщика нагрузки Azure. Подсистема балансировки нагрузки использует пробы для определения, доступны ли виртуальные машины в наборе балансировки нагрузки для приема входящего трафика.

7. Чтобы создать конечную точку балансировки нагрузки, поставьте галочку. Вы увидите **Да** в столбце **Имя набора балансировки нагрузки** на странице **Конечные точки** виртуальной машины.

8. На портале управления щелкните **Виртуальные машины**, выберите имя дополнительной виртуальной машины в наборе балансировки нагрузки, а затем щелкните **Конечные точки** и выберите **Добавить**.

9. На странице **Добавить конечную точку на виртуальную машину** нажмите **Добавить конечную точку в существующий набор балансировки нагрузки**, выберите имя набора балансировки нагрузки и щелкните стрелку вправо.

10. На странице **Ввод сведений о конечной точке** введите имя конечной точки и щелкните флажок.

Для дополнительных виртуальных машин в наборе балансировки нагрузки повторите шаги 8–10.
